# 🏦 Financial Loan Analysis Dashboard

Power BI dashboard for loan data analysis

---

## 📘 Overview

This project presents a comprehensive **3-page Power BI dashboard** designed to analyze financial loan data.  
It provides insights into loan applications, funded amounts, repayment trends, and borrower characteristics.

---

## 📊 Dashboard Pages

### 🔹 1. Overview Dashboard
- High-level KPIs: Total Loan Applications, Funded Amount, Amount Received  
- Monthly trends of funded loans  
- Loan distribution by term (36 vs 60 months)  
- State-wise loan analysis using map visualization  

---

### 🔹 2. Summary Dashboard
- Loan distribution based on employee length  
- Loan purpose analysis (debt consolidation, credit card, etc.)  
- Home ownership-based loan distribution  
- Interactive filters for dynamic data exploration  

---

### 🔹 3. Details Dashboard
- Detailed tabular view of loan records  
- Key attributes: loan amount, interest rate, installment, grade  
- Drill-down analysis for deeper insights  
- Enables granular-level data exploration  

---

## 🛠 Tools & Technologies
- Power BI  
- Power Query  
- DAX  
- Microsoft Excel  

---

## 🔍 Key Features
- Interactive dashboards with slicers and filters  
- KPI tracking (MTD, MoM growth)  
- Data cleaning and transformation using Power Query  
- Custom calculations using DAX  

---

## 💼 Business Impact
- Helps identify loan trends and customer behavior  
- Supports better financial decision-making  
- Enables quick identification of high-performing loan segments
